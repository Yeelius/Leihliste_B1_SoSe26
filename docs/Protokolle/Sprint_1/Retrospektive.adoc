== Ziele der Retrospektive

Einen offenen und konstruktiven Rückblick auf den ersten Sprint zu ermöglichen, um Erkenntnisse zu gewinnen, Verbesserungen zu identifizieren und die Zusammenarbeit im Team zu stärken.

== Stop / Start / Continue

Die Gruppe sammelt Eindrücke und Erkenntnisse von jedem einzelnen Teammitglied, um daraus konkrete Maßnahmen abzuleiten.

=== Stop -> Start
